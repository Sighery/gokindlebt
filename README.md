# GoKindleBT

This is a Go (not pure, requires a C toolchain) API wrapper over [KindleBT][].
The full C documentation is available at the KindleBT repository. For Go
documentation on GoKindleBT, check [pkg.go.dev][gokindlebt pkg.go.dev].

This API is limited to whatever the C API exposes, as well as limitations on
Kindle devices and specific firmware versions. Check the C project for more
information.

[KindleBT]: https://github.com/Sighery/kindlebt
[gokindlebt pkg.go.dev]: https://pkg.go.dev/github.com/Sighery/gokindlebt
